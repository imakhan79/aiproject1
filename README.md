# NeuroVerse AI — Human Potential Operating System

## Stack
- **Client**: React 19 + Vite + TypeScript + Tailwind CSS
- **Server**: Node.js + Express + TypeScript + Prisma
- **Database**: PostgreSQL
- **AI**: Anthropic Claude API (claude-sonnet-4-6) with prompt caching

## Setup

### 1. Prerequisites
- Node.js 18+
- PostgreSQL running locally (or a cloud URL)
- Anthropic API key from https://console.anthropic.com

### 2. Server setup

```bash
cd neuroverse/server

# Copy env file and fill in values
cp .env.example .env
# Edit .env:
#   DATABASE_URL=postgresql://user:password@localhost:5432/neuroverse
#   JWT_SECRET=any-long-random-string
#   ANTHROPIC_API_KEY=sk-ant-...

# Install dependencies
npm install

# Run database migration (creates all tables)
npx prisma migrate dev --name init

# Start server (port 3001)
npm run dev
```

### 3. Client setup

```bash
cd neuroverse/client

# Install dependencies (already done)
npm install

# Start dev server (port 5174)
npm run dev
```

Open http://localhost:5174

## Features

| Feature | Description |
|---------|-------------|
| **Landing Page** | Hero, features overview, CTA |
| **Auth** | Email/password signup & login with JWT |
| **Onboarding** | 7-step wizard: name, age, skills, interests, goals, learning style, personality |
| **AI Future Twin** | Knowledge DNA radar chart, cognitive profile, learning strengths — generated by Claude |
| **Future Simulation** | Pick 2–3 skill paths, Claude simulates salary, probability, timeline for each |
| **AI Life Coach** | Streaming chat powered by Claude, aware of your full profile |
| **Dream Capture** | Voice or text idea input → Claude generates structured 90-day plan + roadmap |
| **Dashboard** | Stats overview, twin preview, quick actions, recent activity |

## API Routes

```
GET          /api/twin

GET          /api/simulation
GET          /api/simulation/:id
POST         /api/simulation

GET          /api/coach/history

GET          /api/dreams
GET          /api/dreams/:id
POST         /api/dreams
PATCH        /api/dreams/:id
DELETE       /api/dreams/:id
```
